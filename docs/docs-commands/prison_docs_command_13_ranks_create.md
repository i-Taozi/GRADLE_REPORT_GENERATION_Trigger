## Description:

Create a rank.

## Permissions:

- `ranks.create`

## SubCommands:

## How to use the command

Execute the command itself following the format like in the example:

- `/ranks create [rankName] [cost] [ladder] [tag]`
- `/ranks create A 100 default &3[&1A&3]&f`
