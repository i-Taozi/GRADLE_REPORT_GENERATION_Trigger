## Description:

Delete a rank.

## Permissions:

- `ranks.delete`

## SubCommands:

- none

## How to use the command

Execute the command itself like in the example to delete a rank:

- `/ranks delete [rankName]`
- `/ranks delete A`

### Command Format

`/ranks delete [rankName]`

**END of the command INFO**
