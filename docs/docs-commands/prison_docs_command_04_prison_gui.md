## Description:

This command will open the `/prison gui` where an admin can manage Prison features.

## Permission:

- `prison.gui`
- `prison.admin`

## SubCommands:

- `none`

## How to use the command

Just execute the command `/prison gui`

### Command Format

`/prison gui`
