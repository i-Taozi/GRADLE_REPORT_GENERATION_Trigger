## Description:

Shows a list of ranks.

## Permissions:

- `ranks.list`

## SubCommands:

- none

## How to use the command

Execute the command itself and add the ladder name you want to see, like in the example: `/ranks list default`.

### Command Format

`/ranks list [ladderName]`

**END of the command INFO**
