## Description:

Get info about a rank.

## Permissions:

- `ranks.info`
- `ranks.admin`

## SubCommands:

- none

## How to use the command

Execute the command like in the example following the format:

- `/ranks info [rankName]`
- `/ranks info A`

### Command Format

`/ranks info [rankName]`

**END of the command INFO**
