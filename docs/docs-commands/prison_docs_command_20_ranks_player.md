## Description:

Shows a player their rank.

## Permission:

- `ranks.admin`

## SubCommands:

- none

## How to use the command

Execute the command itself adding the player Name like in the example: `/ranks player GABRYCA`.

### Command Format

`/ranks player [playerName]`

**END of the command INFO**
